Astronomy

My research interests in astronomy lie in cosmic structure formation. I studied the formation of the cosmic web as it is described by the Zeldovich approximation and the adhesion model.

  • Adhesion example, Python notebooks showing how to compute the adhesion model in 2D.
  • Adhesion model, a literate C++ code for computing the 3D adhesion model using Voronoi tessellations.

Computing

I work at the Netherlands eScience Center, where I collaborate with scientists that work on many different subjects.

Utilities
  • enTangleD, a bi-directional literate programming tool.
  • Pandoc Tangle, a literate programming tool for Pandoc.
  • Noodles, a task-based parallel programming engine for Python.
  • PyXenon, Python interface to the Xenon middleware.
  • Shell-bar, a nifty Bash prompt.
Climate data analysis
  • HyperCanny, a N-dimensional Canny edge detection algorithm, implemented in C++, with Python interface.
  • HyperCC, application of HyperCanny on climate data.

Graphics

I enjoy making fancy graphics.
